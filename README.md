# Tools

This repository contains a collection of tools designed for cybersecurity and ethical hacking.

> [!IMPORTANT]
> **Important**: This repository is intended for ethical and legal use only.

---

## Installation

To install the tools included in this repository, follow the instructions for each specific tool in their respective documentation. Here are the general steps to get started:

1. **Clone this repository**:
	```bash
	git clone https://github.com/HectorPuch/tools.git
	```

2. **Access the directory**:
	```bash
	cd tools
	```

3. **Using the tools**:
	Before running a tool, ensure it has exectuion permissions and then run it:
	```bash
	chmod +x <tool_name>.sh
	./<tool_name>.sh
	```
